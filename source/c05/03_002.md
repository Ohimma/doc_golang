## 3.002 排序: 数组交集

二分查找、排序、双指针  
<https://leetcode-cn.com/problems/intersection-of-two-arrays/>

#### 1. 需求

```
给定两个数组，编写一个函数来计算它们的交集。

输入：nums1 = [1,2,2,1], nums2 = [2,2]
输出：[2]

输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出：[9,4]
```

题解

```
方法一：暴力破解
遍历数组nums1，判断元素是否在数组nums2中，存在则放到新数组中
时间复杂度: O(mn)  // m n 是两个数组长度

方法二：两个集合
使用两个哈希集合存储元素
遍历较小集合
判断是小集合的数据否存在大集合，存在则添加到返回值
时间复杂度：O(m + n)

方法三：排序+双指针
两个数组进行排序
同时遍历两个数组，并作对比
不相等时，较小数字的一方往右移一位
第一次相等时，加入到新切片
第x次相等时，判断是否不等于上个放进去的数据

```

#### 2. go

struct{}是一个无元素的结构体类型，通常在没有信息存储时使用。优点是大小为 0，不需要内存来存储 struct {}类型的值。  
struct{}{}是一个复合字面量，它构造了一个 struct {}类型的值，该值也是空。

```
方法二：两个集合
func diffArr(nums1 []int, nums2 []int) []int {
   set1 := map[int]struct{}{}
    for _, v := range nums1 {
        set1[v] = struct{}{}
    }
    fmt.Println("set1=", set1)
    set2 := map[int]struct{}{}
    for _, v := range nums2 {
        set2[v] = struct{}{}
    }
    fmt.Println("set2=", set2)
    if len(set1) > len(set2) {
        set1, set2 = set2, set1
    }
    var intersection = []int{}
    for v := range set1 {
        // 检查v是否在set2的map中,存在则返回true
        a, has := set2[v]
        fmt.Printf("a=%v  v=%v has=%v\n", a, v, has)
        if has {
            intersection = append(intersection, v)
        }
    }
    return intersection
}

方法三：排序 + 双指针
func diffArr(nums1 []int, nums2 []int) (res []int) {
   sort.Ints(nums1)
   sort.Ints(nums2)
   i, j := 0, 0
   for i < len(nums1) && j < len(nums2) {
       x, y := nums1[i], nums2[j]
       fmt.Printf("x=%v y=%v res=%v\n", x, y, res)
       if x == y {
           if res == nil || x != nums1[i-1] {
              res = append(res, x)
              i++
              j++
           }
       } else if x > y {
           j++
       } else if x < y {
           i++
       }
   }
   return
}
```
